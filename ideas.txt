Proyecto: Drogs+ - Sistema de Gestión para Droguería

Tipo: Aplicación de escritorio en Python (tkinter)
Moneda: Pesos Colombianos (COP)

ARCHIVOS DEL PROYECTO:

1. VenIni.py (Punto de entrada)
   - Ventana de inicio de sesión
   - Campo para ingresar serial de acceso
   - Imagen de cruz azul (logo)
   - Ventana auto-dimensionable

2. VenMenu.py (Menú principal)
   - Navegación entre módulos con 5 botones color-coded
   - Módulos: Inventario, Ventas, Registros, Clientes, Reportes
   - Ventana auto-dimensionable

3. theme.py (Sistema de diseño)
   - Paleta de colores CLARA (modo oscuro eliminado)
   - Funciones helper: crear_header(), crear_card(), crear_boton(),
     crear_entry(), crear_label(), crear_alerta(), crear_treeview()
   - crear_boton() soporta parámetro tamaño ("normal" / "pequeño")
   - Estilos ttk configurados (Treeview, Button, Entry, Scrollbar, Notebook)

4. utils.py
   - Función centrar_ventana() reutilizable

5. Inventario/Inventario.py
   - Formulario para agregar productos (layout grid 2 columnas)
   - Cards: Producto, Fechas y Categoría, Alertas
   - Campos: Nombre, Identificador (auto-generado), Cantidad, Stock Mínimo,
     Precio, Fecha Entrega, Fecha Vencimiento, Categoría, Unidad
   - Categoría por vencimiento: Verde (>12 meses), Amarillo (6-12 meses), Rojo (<6 meses)
   - Alertas de vencimiento con barra lateral de color
   - Alertas de stock mínimo
   - Botones: Volver, Buscar, Editar, Eliminar, Guardar
   - Ventana de eliminación con búsqueda y Treeview
   - Guarda en inventario_productos.xlsx

6. Inventario/Editar_Producto.py
   - Búsqueda de productos con tema visual
   - Edición inline con doble clic
   - Eliminación de productos seleccionados
   - Vista de resultados con colores por categoría (Treeview temático)
   - Ventanas auto-dimensionables

7. Ventas/Ventas.py
   - POS optimizado para facturación rápida
   - Cards: Producto, Detalles del Producto, Cantidad
   - Búsqueda con sugerencias instantáneas
   - Detalles: Stock, Precio, Unidad, Categoría (grid 2x2)
   - Total a pagar con display grande (fondo color primario)
   - Atajos de teclado: Enter para vender
   - Descuento de stock al comprar
   - Registro de cada venta en registros_compra.xlsx
   - Ventana auto-dimensionable

8. Registro.py
   - Visualización de historial de ventas con Treeview temático
   - Filtros: Producto, Fecha, Hora (grid alineado)
   - Paginación compacta al lado derecho (← Página X de Y →)
   - Exportar a Excel, TXT o PDF
   - Eliminación de registros
   - Ventana auto-dimensionable

9. Clientes/Clientes.py
   - Gestión de clientes (CRUD)
   - Campos: Nombre, Cédula, Teléfono, Dirección, Email
   - Treeview temático con colores alternados
   - Guarda en Clientes/clientes.xlsx
   - Ventana auto-dimensionable

10. Reportes/Reportes.py
    - Dashboard con stat cards (ventas, productos, ingresos)
    - Colores en stat cards (azul, amarillo, verde)
    - Ranking de productos más vendidos (Treeview temático)
    - Ventana auto-dimensionable

ARCHIVOS DE DATOS:
- Inventario/inventario_productos.xlsx (stock de productos)
- Ventas/registros_compra.xlsx (historial de ventas)
- Clientes/clientes.xlsx (base de datos de clientes)

=================================================================
CONCEPTO DEL PROYECTO - Visión a Futuro
=================================================================

Drogs+ es una aplicación integral diseñada específicamente para pequeñas farmacias,
con el objetivo de modernizar y profesionalizar la gestión diaria del negocio.

MÓDULOS PRINCIPALES:

1. PUNTO DE VENTA RÁPIDO
   - Ventas fraccionadas (por pastillas, unidades, cajas)
   - Búsqueda instantánea de productos
   - Métodos de pago integrados
   - Facturación electrónica

2. CONTROL DE INVENTARIO INTELIGENTE
   - Alertas automáticas de caducidad (30, 60, 90 días)
   - Niveles mínimos de stock con notificación de reabastecimiento
   - Sugerencia automática de medicamentos genéricos equivalentes
   - Código de barras y lote por producto

3. CUMPLIMIENTO REGULATORIO
   - Registro de recetas para medicamentos controlados
   - Traza completa de compraventa de sustancias reguladas
   - Reportes listos para inspecciones sanitarias

4. FIDELIZACIÓN DE CLIENTES
   - Base de datos de pacientes
   - Recordatorios automáticos de recompra para pacientes crónicos
   - Historial de compras por cliente

5. REPORTES Y ANALÍTICA
   - Rentabilidad por producto, categoría y período
   - Productos más vendidos y menos rotados
   - Dashboard de toma de decisiones
   - Reportes exportables (PDF, Excel)

VISIÓN:
Interfaz sumamente intuitiva que no requiera capacitación compleja,
permitiendo al personal de farmacia enfocarse en lo que importa:
atender bien al paciente.
