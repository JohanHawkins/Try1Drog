Proyecto: Drogs+ - Sistema de Gestión para Droguería

Tipo: Aplicación de escritorio en Python (tkinter)
Moneda: Pesos Colombianos (COP)

ARCHIVOS DEL PROYECTO:

1. VenIni.py (Punto de entrada)
   - Ventana de inicio de sesión
   - Campo para ingresar serial de acceso
   - Imagen de cruz azul (logo)

2. VenMenu.py (Menú principal)
   - Navegación entre módulos con 5 botones color-coded
   - Módulos: Inventario, Ventas, Registros, Clientes, Reportes

3. theme.py (Sistema de diseño)
   - Paleta de colores CLARA
   - Funciones helper: crear_header(), crear_card(), crear_boton(),
     crear_entry(), crear_label(), crear_alerta(), crear_treeview()
   - crear_boton() soporta parámetro tamaño ("normal" / "pequeño")
   - Estilos ttk configurados

4. utils.py
   - Función centrar_ventana() reutilizable

5. Inventario/Inventario.py
   - Formulario layout grid 2 columnas
   - Cards: Producto, Fechas y Categoría, Alertas
   - Alertas de vencimiento con barra lateral de color
   - Botones: Volver, Buscar, Editar, Eliminar, Guardar
   - Ventana de eliminación con búsqueda y Treeview

6. Inventario/Editar_Producto.py
   - Búsqueda y edición inline de productos
   - Colores por categoría en Treeview

7. Ventas/Ventas.py
   - POS con búsqueda instantáneas y sugerencias
   - Detalles del producto en grid 2x2
   - Total a pagar con display grande
   - Atajos de teclado (Enter para vender)

8. Registro.py
   - Historial de ventas con Treeview
   - Filtros: Producto, Fecha, Hora
   - Paginación compacta al lado derecho
   - Exportar a Excel, TXT o PDF

9. Clientes/Clientes.py
   - CRUD de clientes
   - Treeview con colores alternados

10. Reportes/Reportes.py
    - Dashboard con stat cards
    - Ranking de productos más vendidos

ARCHIVOS DE DATOS:
- Inventario/inventario_productos.xlsx
- Ventas/registros_compra.xlsx
- Clientes/clientes.xlsx

RECURSOS:
- images/cruz_azul.ico
- images/cruz_azul.png

DEPENDENCIAS:
- tkinter, pandas, openpyxl, Pillow, tkcalendar, fpdf, python-dateutil

=================================================================
VISIÓN A FUTURO
=================================================================

Drogs+ es una aplicación integral para pequeñas farmacias,
con el objetivo de modernizar la gestión diaria del negocio.

PENDIENTES:

✓ Métodos de pago integrados (Efectivo, Banco [no disponible], Puntos)
✓ Facturación electrónica (PDF + XML estructurado para DIAN)
- Código de barras y lote por producto
- Registro de recetas / cumplimiento regulatorio
- Historial de compras por cliente
- Recordatorios de recompra
